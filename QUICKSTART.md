# Better Twink Mod - Quick Start Guide

## Что это такое?

**Better Twink** - мод-бот для автоматической сортировки предметов в сундуках Minecraft 1.20.1. Бот управляется игроком через удобное меню настроек и автоматически перемещает предметы между выбранными сундуками по настраиваемым правилам.

## Основные возможности

### 1️⃣ Выбор и настройка сундуков
- Откройте меню (клавиша **B**)
- Нажмите "Select Mode" и кликайте на сундуки в мире
- Редактируйте каждый сундук через "Edit Selected"
- Выбранные сундуки подсвечиваются (зеленые линии)

### 2️⃣ Фильтры по модам и предметам
- **Фильтр по модам**: Сундук хранит все предметы из выбранного мода
  - Например: "железные рудины из мода AppliedEnergistics2"
- **Фильтр по предметам**: Конкретные предметы
  - Например: "только железные слитки"
- Можно комбинировать оба фильтра

### 3️⃣ Сундук быстрого сброса (Quick Drop)
- Один или несколько сундуков отмечаются как "Quick Drop Chest"
- В эти сундуки игрок кидает весь лут
- Бот автоматически рассортирует предметы по остальным сундукам

### 4️⃣ Авто-крафт
- Установите правило: "железные блоки → железные слитки"
- Бот будет крафтить слитки в блоки перед складированием
- Настраивается для каждого сундука отдельно

### 5️⃣ Гуманизированные движения
- Настраиваемая задержка между перемещениями (50-1000мс)
- Предотвращает подозрения на использование бота на серверах
- По умолчанию: 100мс

### 6️⃣ Автоматическая навигация
- Бот находит путь между сундуками (A* алгоритм)
- Автоматически открывает двери
- Помнит конфигурацию по серверам

## Первый запуск

### Шаг 1: Установка
```bash
cd e:\minemods\better-twink-mod
./gradlew build
```
Скопируйте `build/libs/bettertwink-1.0.0.jar` в папку `mods/`

### Шаг 2: Запуск игры
- Запустите Minecraft с профилем Forge 1.20.1
- Загрузитесь на сервер

### Шаг 3: Первичная настройка
1. Нажмите **B** (открыть Better Twink Settings)
2. Нажмите **"Select Mode"**
3. Кликайте на сундуки в мире - они добавятся в список
4. Нажмите **"Select Mode"** снова чтобы выйти из режима выбора
5. Выберите сундук в списке и нажмите **"Edit Selected"**

### Шаг 4: Настройка фильтров
В экране редактирования сундука:
- **Rename** - переименуйте сундук (опционально)
- **Quick Drop Chest** - отметьте если это для сброса лута
- **Add Mod Filter** - добавьте фильтр по модам
- **Add Item Filter** - добавьте фильтр по предметам
- **Crafting Rules** - настройте правила крафта (если нужны)

### Шаг 5: Сохранение
- Нажмите **"Save"** после настройки каждого сундука
- После закрытия меню конфигурация автоматически сохраняется

## Примеры использования

### Пример 1: Организация по модам
```
Сундук 1: "AppliedEnergistics" → Все предметы из мода AE2
Сундук 2: "Thermal" → Все предметы из мода Thermal Expansion
Сундук 3: "Minecraft" → Стандартные предметы Minecraft
Сундук 4: "Quick Drop" → Быстрый сброс
```

### Пример 2: Организация по типам материалов
```
Сундук 1: "Железі' → iron_ore, iron_ingot, iron_block
Сундук 2: "Медь" → copper_ore, copper_ingot, copper_block
Сундук 3: "Золото" → gold_ore, gold_ingot, gold_block
Сундук 4: "Quick Drop" → Временный сундук
```

### Пример 3: С авто-крафтом
```
Сундук 1: "Слитки" → Фильтр: iron_ingot
  Правило крафта: iron_ingot → iron_block (конвертация при необходимости)
  
Сундук 2: "Блоки" → Фильтр: iron_block

Сундук 3: "Quick Drop" → Бот сортирует слитки в блоки при необходимости
```

## Файлы конфигурации

Все данные сохраняются в:
```
.minecraft/config/bettertwink/configurations.nbt
```

Структура:
- Отдельная конфигурация для каждого сервера
- Для каждого сервера:
  - Список сундуков с фильтрами
  - Правила крафта
  - Настройки быстрого сброса
  - Флаг авто-сортировки

## Советы для использования

### Производительность
1. **Избегайте больших расстояний** между сундуками
2. **Увеличьте задержку** если сервер замечает подозрительную активность
3. **Используйте быстрый сброс** вместо раздельной сортировки

### Совместимость с серверами
- Мод **полностью client-side** - не требует установки на сервер
- Если сервер запретил быстрое перемещение предметов:
  - Увеличьте `itemTransferDelay` в конфиге
  - Рекомендуется: 300-500мс для большинства серверов

### Оптимизация
1. **Группируйте предметы по модам** а не по типам
2. **Используйте один Quick Drop** сундук на группу сортировочных сундуков
3. **Не создавайте циклические зависимости** в правилах крафта

## Troubleshooting

### Проблема: Бот не движется
**Решение:**
- Проверьте что кнопка "Auto Sort" включена
- Убедитесь что сундуки содержат предметы для сортировки
- Проверьте логи игры на ошибки

### Проблема: Конфигурация не сохраняется
**Решение:**
- Убедитесь что папка `config/bettertwink/` доступна для записи
- Перезагрузитесь на сервер
- Проверьте файл `configurations.nbt` существует

### Проблема: Сундуки не подсвечиваются
**Решение:**
- Убедитесь что включена опция "Highlight Chests" в конфиге
- Проверьте расстояние до сундуков
- Убедитесь что это действительно сундук (CHEST, TRAPPED_CHEST или BARREL)

### Проблема: Правила фильтра не работают
**Решение:**
- Проверьте точное наименование предмета/мода
- Убедитесь что фильтр сохранился (нажмите "Save")
- Перезагрузитесь чтобы применить изменения
