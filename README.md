# Better Twink Mod

**Better Twink** - это клиент-сайдный мод для Minecraft Forge 1.20.1, который автоматизирует сортировку предметов в сундуках.

## Основные функции

- 📦 **Управление сундуками** - Выбирайте сундуки через удобное меню настроек
- 🎨 **Визуализация** - Выбранные сундуки подсвечиваются для удобства
- 🔧 **Фильтры** - Настраивайте, какие типы предметов хранить в каждом сундуке:
  - По модам (все предметы из конкретного мода)
  - По конкретным предметам
- ⚙️ **Автокрафт** - Автоматически конвертируйте предметы по настраиваемым рецептам
- 🚀 **Сундук быстрого сброса** - Выделенный сундук для быстрого выброса предметов, которые бот автоматически рассортирует
- 🤖 **Умная навигация** - Бот находит оптимальный путь между сундуками
- 👤 **Гуманизированные движения** - Настраиваемые задержки для естественного поведения
- 💾 **Сохранение по серверам** - Каждый сервер имеет свою конфигурацию

## Установка

1. Скачайте Minecraft Forge 1.20.1
2. Поместите мод (`.jar` файл) в директорию `mods/`
3. Запустите Minecraft с профилем Forge

## Управление

- **Клавиша B** - Открыть меню настроек Better Twink
- **Левый клик** на сундук в режиме выбора - Выбрать сундук
- **Правый клик** на сундук - Открыть меню конфигурации

## Конфигурация

### Основные параметры (`config/bettertwink/bettertwink-client.toml`)

- `enabled` - Включить/отключить мод
- `itemTransferDelay` - Задержка между перемещениями предметов (в мс)
- `highlightChests` - Подсвечивать выбранные сундуки
- `autoSortEnabled` - Включить автоматическую сортировку

### Сохранение конфигурации

Конфигурация сундуков автоматически сохраняется в `config/bettertwink/configurations.nbt`

Для каждого сервера хранится:
- Позиции выбранных сундуков
- Правила фильтрации
- Правила крафта
- Настройки быстрого сброса

## Архитектура

```
com.minemods.bettertwink/
├── BetterTwinkMod.java          # Главный класс мода
├── config/                       # Конфигурация
├── client/
│   ├── events/                   # Обработчики событий
│   ├── gui/                      # UI экраны
│   ├── render/                   # Визуализация и рендеринг
│   └── storage/                  # Сохранение конфигурации
├── data/                         # Системы данных (ChestConfiguration, ServerConfiguration)
├── sorting/                      # Логика сортировки
├── crafting/                     # Система автокрафта
└── pathfinding/                  # Навигация и поиск пути
```

## Требования

- Minecraft 1.20.1
- Forge 47.2.0+
- Java 17+

## Разработка

Для собственных модификаций используйте:

```bash
./gradlew build          # Собрать мод
./gradlew runClient      # Запустить в девелопер-моде
./gradlew eclipse        # Сгенерировать Eclipse проект
```
